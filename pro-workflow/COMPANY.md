Pro Workflow is an AI coding workflow company built around battle-tested patterns from power users. Each agent operates in a distinct cognitive mode, and work flows through the company as a pipeline:

1. **Orchestrator** decomposes features into Research > Plan > Implement phases with approval gates
2. **Planner** breaks down complex tasks into step-by-step execution plans before any code is written
3. **Scout** runs confidence-gated exploration (0-100 score) and gives GO/HOLD verdicts
4. **Code Reviewer** verifies every finding against actual code — no unverified reports
5. **Debugger** systematically investigates bugs through hypothesize-then-verify methodology
6. **Context Engineer** audits and optimizes context window usage across agents, skills, and MCP servers
7. **Permission Analyst** analyzes denial patterns and generates optimized allow/deny rules
8. **Cost Analyst** identifies expensive token operations and recommends specific optimizations

The philosophy: planning is not implementation, review is not shipping, context optimization is not guessing. Each agent is a specialist with a distinct cognitive mode.
