# Cronicas del Ultimo Hombre — Estudio de Superproduccion

Canal YouTube faceless post-apocaliptico con sistema de produccion de 33 agentes, 8 departamentos y 6 quality gates.

| Metrica | Valor |
|---------|-------|
| Agentes totales | 33 |
| Opus (liderazgo) | 10 |
| Sonnet (especialistas) | 18 |
| Haiku (ejecutores) | 5 |
| Departamentos | 8 |
| Quality Gates | 6 |
| Skills | 14 |
| Duracion episodio | 15 min (14:30-15:30) |

## Pipeline de Produccion

```
FASE 1: Ideacion -----> GATE 1 (Produccion Ejecutiva)
FASE 2: Escritura ----> GATE 2 (Director Creativo + QA Narrativo)
FASE 3: Visual -------> GATE 3 (Director Visual + QA Visual)
FASE 4: Audio/Musica
FASE 5: Produccion ---> GATE 4 (Jefe Postproduccion)
FASE 6: Calidad ------> GATE 5 (Director Calidad)
FASE 7: Distribucion -> GATE 6 (Showrunner CEO)
```

## Version MVP (19 agentes)

Para empezar con una version minima viable, usar solo estos 19 agentes:

showrunner-ceo, produccion-ejecutiva, director-creativo, jefe-de-guion, script-doctor, jefe-narrativa, director-visual, storyboard-artist, ingeniero-prompts-visuales, jefe-postproduccion, supervisor-continuidad, director-calidad, qa-narrativo, qa-final, jefe-distribucion, estratega-youtube, cto, experto-supervivencia, council-presidente

Los demas agentes se fusionan en sus jefes de departamento.

## Stack Tecnologico

- **Hardware**: NVIDIA RTX 5090 (32GB VRAM)
- **Imagenes**: Fooocus (localhost:7860) / ComfyUI (localhost:8188)
- **Video**: LTX Video 2 MAX + Wan2.1
- **Audio**: XTTSv2
- **LLM**: Qwen3.5:35B via Ollama (localhost:11434)
- **Ensamblaje**: FFmpeg
- **Upload**: YouTube API

## Reglas Fundamentales

1. **Coherencia es ley** — narrativa, tonal, visual, emocional
2. **15 minutos sagrados** — tolerancia 14:30-15:30
3. **NEEDS WORK es default** — cada gate rechaza por defecto
4. **Faceless absoluto** — NUNCA se muestra la cara de Elian
5. **Max 3 rechazos** — luego escalacion automatica
6. **Show dont tell** — nunca explicar, siempre mostrar
